== Windows Brain 수동 설치 ==
아래 명령어를 한 줄씩 PowerShell에 붙여넣기

1번:
Invoke-WebRequest -Uri "https://github.com/whynowlab/jarvis-orb/archive/refs/heads/main.zip" -OutFile "$env:TEMP\jarvis-orb.zip"

2번:
Expand-Archive -Path "$env:TEMP\jarvis-orb.zip" -DestinationPath "$env:TEMP\jarvis-orb-extract" -Force

3번:
Copy-Item -Recurse -Force "$env:TEMP\jarvis-orb-extract\jarvis-orb-main\brain\jarvis_brain" "$env:USERPROFILE\.jarvis-orb\jarvis_brain"

4번 (확인):
ls $env:USERPROFILE\.jarvis-orb\jarvis_brain\

5번 (MCP 의존성 설치):
python -m pip install --target "$env:USERPROFILE\.jarvis-orb\lib" mcp

6번 (MCP 테스트):
$env:PYTHONPATH = "$env:USERPROFILE\.jarvis-orb\lib;$env:USERPROFILE\.jarvis-orb"
python -c "from jarvis_brain.mcp_server import mcp; print('OK')"

7번 (Claude Code에 등록):
claude mcp remove jarvis-brain
claude mcp add jarvis-brain -e PYTHONPATH="$env:USERPROFILE\.jarvis-orb\lib;$env:USERPROFILE\.jarvis-orb" -- python -m jarvis_brain.mcp_server
