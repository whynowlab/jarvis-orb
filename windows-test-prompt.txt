아래 작업을 순서대로 실행해줘. 각 단계마다 해당하는 jarvis-brain MCP 도구를 사용해.

1. 기억 저장: "Jarvis Orb v0.3.0 테스트 완료" 라는 내용을 semantic 타입으로 저장해줘 (memory_save)

2. 엔티티 생성: "jarvis-orb" 프로젝트 엔티티를 만들어줘. 상태는 {"status": "testing", "version": "0.3.0"} (entity_create)

3. 엔티티 상태 변경: 방금 만든 엔티티의 상태를 {"status": "verified", "version": "0.3.0"} 으로 변경해줘. 이유는 "Windows 테스트 통과" (entity_update)

4. 기억 검색: "jarvis" 로 검색해줘 (memory_search)

5. 엔티티 조회: project 타입 엔티티를 조회해줘 (entity_query)

6. 관계 생성: 아무 두 엔티티를 "tested_by" 관계로 연결해줘 (entity_relate)

7. 기억 검증: 1번에서 저장한 기억을 verify 해줘 (memory_verify)
